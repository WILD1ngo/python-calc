
Python Calculator

A simple calculator application built with Python 3.7.

Features:
- Basic arithmetic operations (+, -, *, /)
operators:
Advanced Operations:
- Exponentiation (^)
- Modulo (%)
- avg (@)
- min(&)
- max ($)
- sum of the digits (#)
- Factorial (!)
- minus with higher Parentheses (~)
- Parentheses support for complex expressions

Installation 

install Python-3.7

sudo apt install python-3.7

pip install -r requirements.txt


change the settings in calculator.py



to run:
python3.7 .\calculator.py

UI:
I made a simple UI
